# Customer Shopping Behavior Analytics

## Overview

This project analyzes customer shopping behavior using Python, SQL, MySQL, and Power BI to generate business insights and visualize customer purchasing trends.

The project demonstrates an end-to-end analytics workflow including:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* SQL-based Analysis
* Dashboard Visualization

---

## Tech Stack

* Python
* Pandas
* NumPy
* MySQL
* SQLAlchemy
* PyMySQL
* Power BI

---

## Features

* Cleaned and processed customer shopping dataset
* Imported data into MySQL database
* Performed SQL queries for customer trend analysis
* Built interactive Power BI dashboards
* Visualized customer demographics and sales insights

---

## Project Workflow

CSV Dataset → Python Data Cleaning → MySQL Database → SQL Analysis → Power BI Dashboard

---

## Dataset

The dataset contains customer shopping behavior data including:

* Customer ID
* Age
* Gender
* Category
* Purchase Amount
* Payment Method
* Subscription Status
* Frequency of Purchases

## Installation & Setup

### Clone Repository

```bash
git clone https://github.com/bhumika2309/Customer-Shopping-Behavior-Analytics-.git
```

### Install Required Libraries

```bash
pip install pandas numpy sqlalchemy pymysql jupyter
```

### Run Jupyter Notebook

```bash
jupyter notebook
```

---

## Database Setup

Create MySQL database:

```sql
CREATE DATABASE customer_analytics;
```

Update database credentials in the notebook:

```python
username = "root"
password = "your_password"
host = "localhost"
port = "3306"
database = "customer_analytics"
```
